# Frequently Asked Questions (FAQ)

**Q: Do I need to download or install any software to use this?**
A: **No.** This tool operates entirely as a Live Portal. You access it through your web browser just like a normal website, and it works instantly.

**Q: Is it really free? Are there hidden fees?**
A: Yes, it is 100% free and open-source. There are no subscriptions, no premium tiers, and no watermarks.

**Q: How is my privacy protected?**
A: We utilize client-side processing (JavaScript/WebAssembly). When you use the tool, the data is processed directly by your device's CPU/Memory. We have no backend server receiving your files or text. 

**Q: Can I use this tool offline?**
A: Yes! After your first visit to the live portal, the core files are cached in your browser. You can disconnect from the internet and the tool will continue to function.

**Q: Which browsers are supported?**
A: We support all modern browsers including Google Chrome, Mozilla Firefox, Microsoft Edge, and Apple Safari. For the best performance, ensure your browser is updated to the latest version.

**Q: How can I contact the developer?**
