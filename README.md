## 🚀 Featured Projects

Here are some of the projects I am most proud of:

### ✍️ [InkScribe](https://itsrkmahapatra.github.io/InkScribe/)
*Doc se Handwriting Converter.* Turn digital text into realistic handwritten documents offline. It features multiple realistic fonts, customizable ink and paper, and PDF export support. SEO optimized, free, and open-source.

### 🧠 [PromptForge](https://itsrkmahapatra.github.io/PromptForge/)
*Ultimate AI Prompt Generator.* A lightning-fast, client-side tool for generating high-quality prompts for ChatGPT, Midjourney, Claude, and more. 100% free and open-source.

### ⚙️ [api-master](https://github.com/itsrkmahapatra/api-master)
*Comprehensive API Project.* A robust, pre-configured foundation for RESTful services built with Node.js and Express. Features modular routing, security best practices, and ready-to-use middleware.
